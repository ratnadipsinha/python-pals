🐍  PYTHON PALS  —  Learn Python (for kids age 8+)
======================================================

Python Pals is a complete learning app aligned with Kids Python YouTube channel.
11 chapters (132 exercises) + 2 adventure worlds (17 missions).
Real coding practice: kids write actual Python, press ▶ Run to execute, and get
instant feedback with sounds and progress tracking.

KEY FEATURES:
  ✓ Aligned with Kids Python YouTube curriculum
  ✓ 📺 YouTube video links in every chapter
  ✓ 132 hands-on exercises (12 per chapter)
  ✓ Multiple-choice quizzes in Hint pop-ups (never gives the answer)
  ✓ 2 Adventure worlds with empty code boxes (not pre-filled)
  ✓ Progress tracking with % completion + home dashboard
  ✓ 🔄 Reset Progress option
  ✓ Sounds, custom icon, fully scrollable

When you first open the app it asks for your NAME. After that a
HOME screen shows your Progress Summary: the date you started,
today's date, and your % completion bar (exercises + badges).
Press the 🏠 Home button any time to see it again.

From the Home screen you can:
  • ▶ Continue Learning   — go back to your next lesson
  • 🌳 Adventure Mode     — unlock at 90% and play the game worlds
  • ✏️ Change Name         — give yourself a new name
  • 🔄 Reset Progress     — erase ALL progress and start from 0%
                            (your name and start date stay)

--------------------------------------------------
TWO WAYS TO USE IT
--------------------------------------------------

1) PLAY RIGHT NOW (easiest)
   Double-click:  Play-Now.bat
   (Needs Python installed on this PC.)

2) MAKE A REAL WINDOWS APP (.exe, no Python needed to run)
   Double-click:  Build-Windows-App.bat
   It installs what it needs and creates:
       dist\Python Pals.exe
   Copy that single .exe to any Windows 10/11 PC and
   double-click to play — no Python required there.
